# Git Analyzer (Vite + React + Tailwind)
Quick, ready-to-run starter for the GitAnalyzer app (plain Tailwind UI).

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run dev server:
   ```bash
   npm run dev
   ```
3. Open http://localhost:5173

## Notes
- The app uses GitHub public REST API. Unauthenticated requests are rate-limited. To increase limits, create a GitHub personal access token and paste it in the token field in-app.
- If you get CORS or rate errors, try with a token.
